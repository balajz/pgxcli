## Installation

Pick your operating system or preferred installation method:

<Tabs>
  <TabItem label="Linux">
    ### Package Managers

    Pre-built packages are available. Grab the latest from the [releases page](https://github.com/Balaji01-4D/pgxcli/releases).

    **Debian / Ubuntu:**
    ```bash
    sudo dpkg -i pgxcli_*.deb
    ```

    **Fedora / RHEL:**
    ```bash
    sudo rpm -i pgxcli_*.rpm
    ```

    **Alpine:**
    ```bash
    sudo apk add --allow-untrusted pgxcli_*.apk
    ```

    **Arch Linux (AUR):**
    ```bash
    yay -S pgxcli-bin
    ```

    ### Direct Download
    Alternatively, download and extract the binary manually:
    ```bash
    tar -xzf pgxcli_Linux_x86_64.tar.gz
    sudo mv pgxcli /usr/local/bin/
    ```
  </TabItem>
  <TabItem label="Mac">
    pgxcli is available via Homebrew:

    ```bash
    brew install Balaji01-4D/pgxcli/pgxcli
    ```

    Alternatively, grab the latest archive from the [releases page](https://github.com/Balaji01-4D/pgxcli/releases).
  </TabItem>
  <TabItem label="Windows">
    pgxcli is available via Chocolatey:

    ```powershell
    choco install pgxcli
    ```

    Alternatively, grab the latest `.zip` from the [releases page](https://github.com/Balaji01-4D/pgxcli/releases).
  </TabItem>
  <TabItem label="Go Install">
    Make sure you have **Go 1.21+** and a C compiler (CGO is required).

    ```bash
    go install github.com/balaji01-4d/pgxcli/cmd/pgxcli@latest
    ```
  </TabItem>
  <TabItem label="Source Code">
    Make sure you have **Go 1.21+** and a C compiler (CGO is required).

    ```bash
    git clone https://github.com/balaji01-4d/pgxcli.git
    cd pgxcli
    make build
    ```

    The binary lands in `bin/app`. Move it wherever you like (e.g., `/usr/local/bin/pgxcli`).
  </TabItem>
</Tabs>

## Quick Start

Connect to a database:

```bash
pgxcli mydb myuser
```

That's it. You'll get a REPL with syntax highlighting, autocompletion, and history — ready to go.

:::tip
On first run, pgxcli creates a config file at `~/.config/pgxcli/config.toml`. You can customize your prompt, theme, pager, and more. See [Configuration](/pgxcli/guides/configuration/).
:::

### Try a Query

```sql
SELECT version();
```

Results render in a clean table with execution time. If the output is long, pgxcli pages it automatically.

### Exit

Type `\q` or press `Ctrl+C`.
