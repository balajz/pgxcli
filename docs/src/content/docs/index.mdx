## Features

<CardGrid stagger>
	<Card title="Syntax Highlighting" icon="star">
		Queries light up as you type. Powered by Chroma with 10+ built-in themes like `monokai`, `dracula`, and `nord`.
	</Card>
	<Card title="Smart Autocompletion" icon="rocket">
		Tab-complete SQL keywords, table names, columns, functions, and data types — all schema-aware.
	</Card>
	<Card title="Configurable" icon="setting">
		Customize your prompt, highlighting theme, pager behavior, and error handling via a simple TOML config file.
	</Card>
	<Card title="Backslash Commands" icon="list-format">
		Use familiar PostgreSQL commands like `\d`, `\dt`, `\l`, `\c`, and more — right in your REPL.
	</Card>
</CardGrid>
